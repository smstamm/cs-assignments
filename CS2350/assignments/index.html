Steph's Assignments

Learning Activities explore new concepts under the guidance of the instructor where the student adds finishing touches to an existing project.

Try It Out Activities implement the concepts explored in the learning activity. A project is provided to build on, but guided instruction is not provided.

Challenge Activities allow the student to implement the concepts learned with full creative freedom. No guided instruction or project is provided to build on.
